# terminal-profile

Небольшой сайт-портфолио в виде терминала.

## Русский

Это простой интерактивный терминал для GitHub Pages. При открытии он запускает `neofetch`, показывает стек, контакты и публичные репозитории GitHub.

Что есть:

- перетаскиваемое окно
- изменение размера окна
- команды `help`, `neofetch`, `projects`, `stack`, `contact`, `snake`, `clear`
- змейка прямо в терминале
- автоязык: русский для СНГ, английский для остальных

Чтобы поменять данные, открой `script.js` и измени объект `profile`.

## English

A small terminal-style portfolio site.

It runs a fake `neofetch` on load, shows stack, contact info, and public GitHub repositories.

Features:

- draggable window
- resizable window
- commands: `help`, `neofetch`, `projects`, `stack`, `contact`, `snake`, `clear`
- snake game inside the terminal
- auto language: Russian for CIS region, English for everyone else

To change profile data, edit the `profile` object in `script.js`.
